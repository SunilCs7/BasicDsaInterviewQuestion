

### ✅ **Top Basic DSA Interview Questions (Development Roles ke liye)**

#### 📌 **Array & Strings:**
1. **Find the duplicate in an array**  
2. **Find the missing number in a sequence**  
3. **Reverse an array/string**  
4. **Check if a string is a palindrome**  
5. **Move all zeros to end of array**   
6. **Kadane’s Algorithm (Maximum Subarray Sum)**  
7. **Two Sum problem** (Find two numbers that add up to a target)

#### 📌 **Sorting & Searching:**
8. **Bubble sort / Selection sort / Insertion sort**
9. **Binary Search** (Iterative + Recursive)
10. **Sort 0s, 1s, and 2s** (Dutch National Flag)

#### 📌 **Hashing / Maps / Sets:**
11. **First non-repeating character in a string**
12. **Count frequency of elements in an array**
13. **Check for anagram**

#### 📌 **Linked List (Basic only):**
14. **Reverse a linked list**
15. **Detect a loop in a linked list** (Floyd's algorithm)

#### 📌 **Stacks & Queues:**
16. **Valid Parentheses** (Using Stack)
17. **Implement stack using array or vector**

#### 📌 **Recursion (Simple):**
18. **Factorial of a number**
19. **Fibonacci number**
20. **Print all subsets of a string/array**

---

### 🎯 Interview Tips:
- Start answers with: **“I’ll first explain the approach”**
- Time complexity aur space complexity zarur mention karna.
- Agar coding karne ko bole, toh pehle dry-run kar lena 1 test case pe.
- Agar stuck ho jaaye toh soch kar bata, turant give up mat kar.
